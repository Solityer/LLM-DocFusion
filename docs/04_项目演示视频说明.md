## 三、录制注意事项

### 3.1 录制前准备

- [ ] 确认 Ollama 服务已启动，`ollama list` 可见 qwen2.5:14b
- [ ] 确认 `./start.sh` 已执行，系统正常运行
- [ ] 关闭不相关的浏览器标签页、通知弹窗
- [ ] 确认音频输入（麦克风）正常
- [ ] 提前准备好测试文件路径，避免演示中找文件耽误时间

### 3.2 推荐录制流程

1. 先按脚本完整走一遍，记录实际耗时
2. 确认所有操作结果符合预期
3. 正式录制时可适当加速（×1.5倍速）较慢的 LLM 处理等待阶段
4. 后期剪辑中可在 LLM 处理阶段叠加倒计时或进度提示

### 3.3 常见问题排查

| 问题 | 解决方案 |
|:----:|:--------:|
| Ollama 未响应 | 执行 `ollama serve`，检查端口 11434 |
| 文件上传失败 | 检查文件格式是否在白名单内 |
| 处理超时 | CPU 模式下 qwen2.5:14b 较慢，可适当剪辑视频 |
| 前端无法访问 | 确认 `./start.sh` 输出的实际端口号 |
