## API 端点汇总

### 原有端点（保持兼容）

| 端点 | 说明 |
|-----|------|
| `GET /api/health` | 模型健康检查 |
| `POST /api/files/upload` | 上传数据源文件 |
| `POST /api/templates/upload` | 上传模板文件 |
| `POST /api/process/local` | 启动本地文件处理任务 |
| `GET /api/task/{task_id}` | 查询任务状态 |
| `POST /api/sources/preview` | 预览数据源 |
| `POST /api/document/operate` | 文档智能操作 |

### 新增端点（v2.0）

| 端点 | 说明 |
|-----|------|
| `POST /api/store/import/upload` | 上传文档入库 |
| `POST /api/store/import/local` | 本地路径入库 |
| `GET /api/store/status/{task_id}` | 入库任务状态 |
| `GET /api/store/documents` | 获取已入库文档列表 |
| `GET /api/store/documents/{id}` | 文档详情 |
| `GET /api/store/search?q=` | 全文检索 |
| `GET /api/store/entities` | 提取实体列表 |
| `GET /api/store/quality` | 质量问题列表 |
| `GET /api/store/stats` | 数据库统计 |
| `GET /api/analytics/dashboard` | 分析看板数据 |
| `GET /api/report/{task_id}/markdown` | Markdown 竞赛报告 |
| `POST /api/evaluate/compare` | 评估对比（JSON） |
| `POST /api/evaluate/compare/upload` | 评估对比（文件上传） |
| `POST /api/templates/inspect/upload` | 模板字段预览（上传） |
| `POST /api/templates/inspect/local` | 模板字段预览（本地路径） |
